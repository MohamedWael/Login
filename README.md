## Integration

1- Add it in your root build.gradle at the end of repositories:

<pre>
	allprojects {
		repositories {
			...
		  maven { url 'https://jitpack.io' }
		}
	}
</pre>

2- Add the dependency

<pre>
dependencies {
    ...
    <b>implementation 'com.github.MohamedWael:Login-library:1.0.0'</b>
}
</pre>
